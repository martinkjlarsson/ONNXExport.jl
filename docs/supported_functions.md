# Supported Julia functions
The following Julia functions are supported in some capacity:

* Array operations: fill, reshape, vec, transpose, adjoint, permutedims, repeat, cat, hcat, vcat, hvcat, accumulate, cumprod, cumsum, dropdims, partialsort, reverse
* Array math: +, -, *, inv
* Control flow: ifelse
* Indexing: getindex, selectdim
* Linear algebra: det, dot, I, tril, triu, norm, opnorm, normalize, kron
* Math: acosh, asin, asinh, atan, atanh, cos, cosh, exp, log, inv, sin, sinh, sqrt, tan, tanh, abs, sign, ~, !, ceil, floor, round, isinf, isnan, +, -, *, /, ^, max, min, mod, &, |, xor, <<, >>, >>>, ==, !=, >, >=, <, <=, clamp
* Reductions: unique, maximum, minimum, extrema, argmax, argmin, sum, prod, any, all, count
* Statistics: mean

## SpecialFunctions.jl extension
* Error functions: erf
